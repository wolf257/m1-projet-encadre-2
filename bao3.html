Extraction de patrons morphosyntaxique en perl.

Objectif et déroulement :

La BaO3 se fonde sur les résultats de la BaO2. Elle permet l'extraction, dans les fichiers Talismane générés par la BaO2, de patrons morpho-syntaxiques qui se trouvent en ligne dans le fichier patrons.txt ( voir ).

Le résultat de l'extraction est écrit dans un fichier txt, dans lequel se trouve également le nombre total de matchs.

Pour chaque patron, la liste des correspondances est triée par ordre décroissant de fréquence.
